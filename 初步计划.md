# 全原子蛋白质设计模型：完整实现方案

## 项目概述

基于 Rectified Flow Matching + 非等变 DiT + atom14 表征的全原子蛋白质生成模型，支持 EC 化学信息条件生成与 motif-scaffolding。

**核心设计原则**：
- 不使用 IPA、Frame、Rigid Group、χ角重建
- 不使用等变网络层，等变性完全靠训练时 SE(3) 数据增强
- 不处理小分子/核酸，只做纯蛋白质
- 使用标准 Transformer + FlashAttention，最大化训练和推理效率

### 3.2 SE(3) 数据增强

### `data/transforms.py`

**这是非等变方案的命脉。** 每个样本每次取出时，施加随机的旋转+平移。

```python
def random_se3_augmentation(coords: Tensor) -> Tensor:
    """
    coords: [N_res, 14, 3]
    
    1. 随机旋转: 均匀采样 SO(3) 旋转矩阵
    2. 随机平移: 先将质心移到原点，再加小随机平移
    """
    # 质心归零（重要！不归零的话平移范围太大）
    ca_coords = coords[:, 1, :]  # index 1 = Cα
    centroid = ca_coords.mean(dim=0)
    coords = coords - centroid[None, None, :]
    
    # 随机 SO(3) 旋转
    R = random_rotation_matrix()  # [3, 3], 均匀采样
    coords = torch.einsum('ij, bnj -> bni', R, coords)
    
    return coords


def random_rotation_matrix() -> Tensor:
    """均匀采样 SO(3) 旋转矩阵 (QR分解法)"""
    H = torch.randn(3, 3)
    Q, R = torch.linalg.qr(H)
    Q = Q @ torch.diag(torch.sign(torch.diag(R)))
    if torch.det(Q) < 0:
        Q[:, 0] *= -1
    return Q
```

**RBF 编码**（将连续距离转为多维特征）：
```python
def rbf_encode(distances, d_min=0.0, d_max=22.0, num_rbf=16):
    """
    distances: [...]
    返回: [..., num_rbf]
    
    用等间距高斯核编码距离
    """
    mu = torch.linspace(d_min, d_max, num_rbf)  # [num_rbf]
    sigma = (d_max - d_min) / num_rbf
    return torch.exp(-0.5 * ((distances.unsqueeze(-1) - mu) / sigma) ** 2)
```

### 3.4 EC 层级向量预计算

### `data/ec_precompute.py`

**训练前一次性运行，预计算所有 EC 编号在各层级截断下的 Qwen3 向量。**

EC 条件采用**层级 CFG dropout** 策略：rxn（反应方程式）始终保留作为核心化学锚点，EC 层级逐步叠加提供从粗到细的折叠先验。

```python
EC 层级文本构造规则:

原始数据格式 (你的JSON):
  "1.1.1.1": "EC 1.1.1.1 | Oxidoreductases | Acting on the CH-OH group of donors 
               | With NAD(+) or NADP(+) as acceptor | alcohol dehydrogenase 
               | Reactions: a primary alcohol + NAD(+) = an aldehyde + NADH + H(+)"

拆解为组件:
  lv1  = "Oxidoreductases"
  lv2  = "Acting on the CH-OH group of donors"
  lv3  = "With NAD(+) or NADP(+) as acceptor"
  name = "alcohol dehydrogenase"
  rxn  = "Reactions: a primary alcohol + NAD(+) = an aldehyde + NADH + H(+)"

构造 6 个层级的文本:
  Level 0 (无条件):    ""
  Level 1 (rxn-only):  "{rxn}"
  Level 2 (lv1+rxn):   "{lv1} | {rxn}"
  Level 3 (lv1+2+rxn): "{lv1} | {lv2} | {rxn}"
  Level 4 (lv1+2+3+rxn): "{lv1} | {lv2} | {lv3} | {rxn}"
  Level 5 (full):      "{lv1} | {lv2} | {lv3} | {name} | {rxn}"

设计理念:
  - rxn 始终保留: 反应方程式是最硬的化学约束 (底物、辅因子、断键方式)
  - EC 层级逐步叠加: 提供从粗到细的折叠先验
  - 推理时可用层级 CFG 分别控制化学反应引导和折叠类型引导的强度
"""

import json
import torch
from tqdm import tqdm

NUM_EC_LEVELS = 6  # 0=无条件, 1=rxn, 2=lv1+rxn, 3=lv1+2+rxn, 4=lv1+2+3+rxn, 5=full


def parse_ec_description(ec_number: str, description: str) -> dict:
    """
    将原始 EC 描述文本拆解为层级组件。
    
    返回:
    {
        'ec_number': '1.1.1.1',
        'lv1': 'Oxidoreductases',
        'lv2': 'Acting on the CH-OH group of donors',
        'lv3': 'With NAD(+) or NADP(+) as acceptor',
        'name': 'alcohol dehydrogenase',
        'rxn': 'Reactions: a primary alcohol + NAD(+) = an aldehyde + NADH + H(+)',
    }
    """
    # 按 " | " 分割, 跳过第一段 "EC x.x.x.x"
    parts = description.split(" | ")
    # parts[0] = "EC 1.1.1.1", parts[1] = "Oxidoreductases", ...
    
    # 找到 Reactions 的位置
    rxn_idx = None
    for i, p in enumerate(parts):
        if p.strip().startswith("Reactions:"):
            rxn_idx = i
            break
    
    return {
        'ec_number': ec_number,
        'lv1': parts[1] if len(parts) > 1 else "",
        'lv2': parts[2] if len(parts) > 2 else "",
        'lv3': parts[3] if len(parts) > 3 else "",
        'name': parts[4] if len(parts) > 4 and rxn_idx and rxn_idx > 4 else "",
        'rxn': " | ".join(parts[rxn_idx:]) if rxn_idx else "",
    }


def build_level_texts(components: dict) -> dict:
    """
    从组件构造 6 个层级的文本。

        components['lv1'], components['lv2'], components['lv3'],
        components['name'], components['rxn']
    )
    
    texts = {
        0: "",                                                              # 无条件
        1: rxn,                                                             # rxn-only
        2: f"{lv1} | {rxn}" if lv1 else rxn,                               # lv1+rxn
        3: f"{lv1} | {lv2} | {rxn}" if lv2 else f"{lv1} | {rxn}",          # lv1+2+rxn
```

```python
        'atom_mask':     [N_res, 14],         # 原子mask
        'aatype':        [N_res],             # 氨基酸类型 (scaffold区域设为20=mask)
        'residue_index': [N_res],             # 残基编号
        'pair_features': [N_res, N_res, d],   # pair初始特征
        'ec_vector':     [d_qwen],            # 当前层级的 Qwen3 向量
        'ec_level':      scalar (int),        # 当前层级 0-5
        'motif_mask':    [N_res],             # 1=motif残基, 0=scaffold
    }
    """
    
    def __init__(self, pdb_dir, ec_vectors_path=None, max_len=256, motif_prob=0.5):
        # ... 加载蛋白质数据 ...
        self.motif_prob = motif_prob
        
        # 加载预计算的 EC 层级向量
        if ec_vectors_path:
            ec_data = torch.load(ec_vectors_path)
            self.ec_vectors = ec_data['vectors']   # dict: (ec_number, level) → tensor
            self.d_qwen = ec_data['d_qwen']
        else:
            self.ec_vectors = None
    
    def __getitem__(self, idx):
        # ... 加载蛋白质结构 ...
        
        # SE(3) 数据增强
        coords = random_se3_augmentation(coords)
        
        # === EC 层级 CFG dropout ===
        if self.ec_vectors is not None:
            ec_number = self.ec_numbers[idx]
            level = random.choices(
                [0,    1,    2,    3,    4,    5   ],
                weights=[0.10, 0.10, 0.15, 0.15, 0.20, 0.30],
            )[0]
            ec_vector = self.ec_vectors.get(
                (ec_number, level),
                torch.zeros(self.d_qwen)
            )
        else:
            ec_vector = torch.zeros(self.d_qwen or 1024)
            level = 0
        
        # === Motif-scaffolding (50%概率启用) ===
        N = coords.shape[0]
        aatype_original = self.aatypes[idx]  # [N]
        
        if random.random() < self.motif_prob:
            n_motif = random.randint(2, min(8, N // 4))
            motif_indices = random.sample(range(N), n_motif)
            motif_mask = torch.zeros(N)
            motif_mask[motif_indices] = 1.0
            
            # motif 区域保留真实 aatype, scaffold 设为 mask token
            aatype = aatype_original.clone()
            aatype[motif_mask == 0] = 20  # mask token
        else:
            motif_mask = torch.zeros(N)
            aatype = torch.full((N,), 20, dtype=torch.long)  # 全部 mask
        
        return {
            'coords': coords,
            'atom_mask': atom_mask,
            'aatype': aatype,
            'residue_index': residue_index,
            'ec_vector': ec_vector,
            'ec_level': level,
            'motif_mask': motif_mask,
        }
```

**层级 dropout 概率分配说明**：
```
Level 0 (无条件):      10%  → CFG 的 baseline
Level 1 (rxn-only):    10%  → 学会仅从反应方程式引导结构
Level 2 (lv1+rxn):     15%  → 学会大类折叠先验 + 反应
Level 3 (lv1+2+rxn):   15%  → 学会亚类 + 反应
Level 4 (lv1+2+3+rxn): 20%  → 学会具体机制 + 反应
Level 5 (full):         30%  → 完整信息, 最高频 (保证模型见足够多完整条件)
```

**数据预处理建议**：
- 先用 `pdb_parser.py` 将所有 PDB 转为 `.pt` 文件，训练时直接 load，避免反复解析
- 按链长分桶（bucketing），同一 batch 内链长相近，减少 padding 浪费
- 预计算 pair_features 中不依赖 SE(3) 增强的部分（如 rel_pos）

## 四、模型架构

### 4.1 总体流程

```
输入:
  x_t:           [B, N_res, 14, 3]    噪声坐标
  t:             [B]                  时间步 (0~1)
  atom_mask:     [B, N_res, 14]       原子mask
  aatype:        [B, N_res]           氨基酸类型 (仅motif区域有, scaffold区域可设为特殊token)
  residue_index: [B, N_res]           残基编号
  ec_vector:     [B, d_ec]            EC条件向量
  motif_mask:    [B, N_res]           1=motif, 0=scaffold (仅scaffolding任务)

流程:
  1. Atom Encoder:     [B, N_res, 14, 3] → [B, N_res, d_token]
  2. Pair Init:        → [B, N_res, N_res, d_pair]
  3. Pairformer ×N:    更新 token_repr 和 pair_repr
                       条件: ec_vector + t → AdaLN
  4. Atom Decoder:     [B, N_res, d_token] → [B, N_res, 14, 3]

输出:
  v_theta:       [B, N_res, 14, 3]    预测的速度场
```

### 4.2 Atom Encoder

### `model/atom_encoder.py`

```python
class AtomEncoder(nn.Module):
    """
    将每个残基的 14 个原子坐标编码为一个 d_token 维的 token 向量。
    
    输入:
        atom_coords:  [B, N_res, 14, 3]
        atom_mask:    [B, N_res, 14]
        aatype:       [B, N_res]         (可选, motif区域)
    
    输出:
        token_repr:   [B, N_res, d_token]
        atom_repr:    [B, N_res, 14, d_atom]  (保留, decoder要用)
    
    内部结构:
        1. 原子输入嵌入:
           - atom_coords:             3维
           - atom_position_embed:     14个位置的可学习嵌入 (标记这是第几个原子)
           - atom_element_embed:      元素类型嵌入 (C/N/O/S)
           - atom_mask:               1维 (告诉模型哪些是真实原子)
           拼接后 → Linear → d_atom 维
        
        2. 残基内注意力 (Intra-Residue Attention):
           - 3层标准 Transformer
           - 序列长度仅为 14 (非常小, 不需要FlashAttention)
           - attention mask = atom_mask (虚拟原子不参与)
        
        3. 聚合:
           - 取 index=1 (Cα) 位置的特征作为 token_repr
           - 或者对所有真实原子做 masked mean pooling
           → Linear → d_token 维
    """
    
    def __init__(self, d_atom=128, d_token=256, n_layers=3, n_heads=4):
        super().__init__()
        self.atom_position_embed = nn.Embedding(14, d_atom)
        self.atom_element_embed = nn.Embedding(4, d_atom)  # C, N, O, S
        self.input_proj = nn.Linear(3 + 1, d_atom)  # coords(3) + mask(1)
        self.combine = nn.Linear(d_atom * 3, d_atom)  # 拼接三个嵌入
        
        self.intra_attn = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_atom, n_heads, d_atom * 4, batch_first=True),
            num_layers=n_layers
        )
        
        self.to_token = nn.Linear(d_atom, d_token)
    
    def forward(self, atom_coords, atom_mask, aatype=None):
        B, N, A = atom_coords.shape[:3]  # A=14
        
        # 1. 原子特征
        coord_feat = self.input_proj(
            torch.cat([atom_coords, atom_mask.unsqueeze(-1)], dim=-1)
        )                                                    # [B, N, 14, d_atom]
        pos_embed = self.atom_position_embed(
            torch.arange(14, device=atom_coords.device)
        )                                                    # [14, d_atom]
        # element_embed 需要查表: atom14 index → element type
        # (预计算存在 constants.py 中)
        
        atom_feat = self.combine(
            torch.cat([coord_feat, pos_embed.expand(B,N,-1,-1), ...], dim=-1)
        )                                                    # [B, N, 14, d_atom]
        
        # 2. 残基内注意力 (reshape 为 [B*N, 14, d_atom])
        atom_feat = atom_feat.reshape(B*N, 14, -1)
        attn_mask = (atom_mask.reshape(B*N, 14) == 0)       # True=忽略
        atom_feat = self.intra_attn(atom_feat, src_key_padding_mask=attn_mask)
        atom_feat = atom_feat.reshape(B, N, 14, -1)          # [B, N, 14, d_atom]
        
        # 3. 聚合: 取 Cα 特征
        token_repr = self.to_token(atom_feat[:, :, 1, :])    # [B, N, d_token]
        
        return token_repr, atom_feat
```

### 4.3 Pair Representation 初始化

### `model/pair_repr.py`

```python
class PairRepresentationInit(nn.Module):
    """
    从几何特征和 token_repr 初始化 pair representation。
    
    输入:
        token_repr:    [B, N, d_token]
        residue_index: [B, N]
        ca_coords:     [B, N, 3]        (从atom_coords[:,:,1,:]取)
    
    输出:
        pair_repr:     [B, N, N, d_pair]
    
    组成:
        1. 相对位置编码:
           rel_pos = clamp(j - i, -32, 32)  → one_hot(65维) → Linear → d_pair
        
        2. Cα 距离 RBF:
           dist = ||ca_i - ca_j||  → rbf_encode(16维) → Linear → d_pair
        
        3. 外积 (outer product):
           token_repr → Linear → [B, N, d_proj]
           pair_outer = einsum('bid, bjd -> bijd')  → Linear → d_pair
        
        4. 求和:
           pair_repr = rel_pos_feat + dist_feat + pair_outer
    """
    
    def __init__(self, d_token=256, d_pair=128, num_rbf=16):
        super().__init__()
        self.rel_pos_embed = nn.Linear(65, d_pair)
        self.dist_embed = nn.Linear(num_rbf, d_pair)
        self.proj_left = nn.Linear(d_token, d_pair)
        self.proj_right = nn.Linear(d_token, d_pair)
        self.outer_proj = nn.Linear(d_pair, d_pair)
```

### 4.4 Pairformer Block（核心）

### `model/pairformer.py`

```python
class PairformerBlock(nn.Module):
    """
    一个 Pairformer block, 包含:
    
    1. Token Track:
       - AdaLN(token_repr, condition)
       - Pair-Biased Self-Attention
       - AdaLN → SwiGLU FFN
    
    2. Pair Track (可选, 建议先不加三角更新):
       - Pair transition (两层MLP)
       - (可选) Triangle multiplicative update
       - (可选) Triangle attention
    
    Pair-Biased Attention 的做法:
       - 标准多头注意力: Q, K, V 来自 token_repr
       - pair_repr → Linear → [B, N, N, n_heads]  作为 attention logits 的 bias
       - attention_weights = softmax(QK^T / sqrt(d) + pair_bias)
    
    这是 AF2/AF3 成对表征影响注意力的核心机制。
    """
    
    def __init__(self, d_token, d_pair, n_heads, d_cond):
        super().__init__()
        
        # AdaLN-Zero
        self.adaln1 = AdaLNZero(d_token, d_cond)
        self.adaln2 = AdaLNZero(d_token, d_cond)
        
        # Pair-biased attention
        self.q_proj = nn.Linear(d_token, d_token)
        self.k_proj = nn.Linear(d_token, d_token)
        self.v_proj = nn.Linear(d_token, d_token)
        self.pair_bias_proj = nn.Linear(d_pair, n_heads)  # pair → bias
        self.out_proj = nn.Linear(d_token, d_token)
        
        # FFN
        self.ffn = SwiGLU_FFN(d_token, d_token * 4)
        
        # Pair transition (简单两层MLP)
        self.pair_transition = nn.Sequential(
            nn.LayerNorm(d_pair),
            nn.Linear(d_pair, d_pair * 4),
            nn.SiLU(),
            nn.Linear(d_pair * 4, d_pair),
        )
        
        self.n_heads = n_heads
    
    def forward(self, token_repr, pair_repr, cond):
        """
        token_repr: [B, N, d_token]
        pair_repr:  [B, N, N, d_pair]
        cond:       [B, d_cond]   (ec_vector + time_embed 合并后)
        """
        # === Token Track ===
        
        # 1. Pair-biased self-attention with AdaLN
        h, gate1 = self.adaln1(token_repr, cond)  # h是norm后的, gate1是残差门控
        
        Q = self.q_proj(h).reshape(B, N, self.n_heads, -1).permute(0,2,1,3)
        K = self.k_proj(h).reshape(B, N, self.n_heads, -1).permute(0,2,1,3)
        V = self.v_proj(h).reshape(B, N, self.n_heads, -1).permute(0,2,1,3)
        
        # pair bias: [B, N, N, d_pair] → [B, N, N, n_heads] → [B, n_heads, N, N]
        pair_bias = self.pair_bias_proj(pair_repr).permute(0, 3, 1, 2)
        
        # 注意力 (可替换为 FlashAttention with bias)
        attn = (Q @ K.transpose(-1, -2)) / math.sqrt(Q.shape[-1])
        attn = attn + pair_bias
        attn = F.softmax(attn, dim=-1)
        
        out = (attn @ V).permute(0,2,1,3).reshape(B, N, -1)
        out = self.out_proj(out)
        
        token_repr = token_repr + gate1 * out
        
        # 2. FFN with AdaLN
        h, gate2 = self.adaln2(token_repr, cond)
        token_repr = token_repr + gate2 * self.ffn(h)
        
        # === Pair Track ===
        pair_repr = pair_repr + self.pair_transition(pair_repr)
        
        return token_repr, pair_repr
```

### 4.5 AdaLN-Zero

### `model/adaln.py`

```python
class AdaLNZero(nn.Module):
    """
    Adaptive Layer Normalization with Zero initialization.
    
    输入:
        x:    [B, N, d]     待归一化的特征
        cond: [B, d_cond]   条件向量 (ec_vector + time_embed)
    
    输出:
        h:    [B, N, d]     归一化+调制后的特征
        gate: [B, 1, d]     残差门控 (初始化为0)
    
    公式:
        gamma, beta, alpha = Linear(SiLU(Linear(cond)))  # 各 d 维
        h = LayerNorm(x) * (1 + gamma) + beta
        gate = alpha
    
    初始化:
        最后一层 Linear 的权重和偏置全部初始化为 0
        → alpha 初始为 0 → 残差连接初始为恒等映射
        → 加条件不会破坏预训练好的特征
    """
    
    def __init__(self, d_model, d_cond):
        super().__init__()
        self.norm = nn.LayerNorm(d_model)
        self.proj = nn.Sequential(
            nn.Linear(d_cond, d_cond),
            nn.SiLU(),
            nn.Linear(d_cond, d_model * 3),  # gamma, beta, alpha
        )
        # Zero-init 最后一层
        nn.init.zeros_(self.proj[-1].weight)
        nn.init.zeros_(self.proj[-1].bias)
    
    def forward(self, x, cond):
        gamma, beta, alpha = self.proj(cond).unsqueeze(1).chunk(3, dim=-1)
        h = self.norm(x) * (1 + gamma) + beta
        return h, alpha
```

### 4.6 Atom Decoder

### `model/atom_decoder.py`

```python
class AtomDecoder(nn.Module):
    """
    将 token_repr 解码回 atom14 坐标 (速度场)。
    基本上是 AtomEncoder 的镜像操作。
    
    输入:
        token_repr: [B, N, d_token]
        atom_repr:  [B, N, 14, d_atom]   (encoder保留的中间特征)
        atom_mask:  [B, N, 14]
    
    输出:
        v_atom14:   [B, N, 14, 3]        (预测的速度场)
    
    内部结构:
        1. token_repr → Linear → d_atom → 广播到14个原子
        2. 与 encoder 的 atom_repr 相加 (skip connection)
        3. 残基内注意力 × 3层
        4. Linear → 3维 (每个原子的速度向量)
    """
    
    def __init__(self, d_atom=128, d_token=256, n_layers=3, n_heads=4):
        super().__init__()
        self.token_to_atom = nn.Linear(d_token, d_atom)
        self.intra_attn = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_atom, n_heads, d_atom * 4, batch_first=True),
            num_layers=n_layers
        )
        self.to_coords = nn.Linear(d_atom, 3)
    
    def forward(self, token_repr, atom_repr, atom_mask):
        B, N = token_repr.shape[:2]
        
        # 1. 广播 token → 14 原子
        token_broadcast = self.token_to_atom(token_repr).unsqueeze(2).expand(-1,-1,14,-1)
        
        # 2. 与 encoder 中间特征相加
        atom_feat = atom_repr + token_broadcast  # [B, N, 14, d_atom]
        
        # 3. 残基内注意力
        atom_feat = atom_feat.reshape(B*N, 14, -1)
        attn_mask = (atom_mask.reshape(B*N, 14) == 0)
        atom_feat = self.intra_attn(atom_feat, src_key_padding_mask=attn_mask)
        atom_feat = atom_feat.reshape(B, N, 14, -1)
        
        # 4. 输出速度
        v = self.to_coords(atom_feat)  # [B, N, 14, 3]
        
        # 虚拟原子的速度置零
        v = v * atom_mask.unsqueeze(-1)
        
        return v
```

### 4.7 整合模型

### `model/flow_model.py`

```python
class AllAtomFlowModel(nn.Module):
    """
    完整模型, 整合所有子模块。
    
    超参数 (base config):
        d_token:    256
        d_pair:     128
        d_atom:     128
        d_cond:     256    (EC向量+时间嵌入合并后)
        n_trunk_layers: 12  (Pairformer block数)
        n_atom_layers:  3   (encoder/decoder各3层)
        n_heads:    8
        d_qwen:     1024   (Qwen3-embedding 输出维度)
    """
    
    def __init__(self, config):
        super().__init__()
        
        # 时间步嵌入
        self.time_embed = SinusoidalPosEmbed(config.d_cond // 2)
        self.time_proj = nn.Sequential(
            nn.Linear(config.d_cond // 2, config.d_cond),
            nn.SiLU(),
            nn.Linear(config.d_cond, config.d_cond),
        )
        
        # EC 条件投影: Qwen3 向量 → 瓶颈 → 条件维度
        # 瓶颈层强制压缩, 逼掉 Qwen3 中与蛋白质结构无关的语言学噪声
        self.ec_proj = nn.Sequential(
            nn.Linear(config.d_qwen, 128),     # 1024 → 128 大幅降维
            nn.SiLU(),
            nn.Linear(128, 64),                 # 128 → 64 瓶颈
            nn.SiLU(),
            nn.Linear(64, config.d_cond),       # 64 → 256 映射到条件空间
        )
        
        # 子模块
        self.atom_encoder = AtomEncoder(config.d_atom, config.d_token, config.n_atom_layers)
        self.pair_init = PairRepresentationInit(config.d_token, config.d_pair)
        self.trunk = nn.ModuleList([
            PairformerBlock(config.d_token, config.d_pair, config.n_heads, config.d_cond)
            for _ in range(config.n_trunk_layers)
        ])
        self.atom_decoder = AtomDecoder(config.d_atom, config.d_token, config.n_atom_layers)
    
    def forward(self, x_t, t, atom_mask, aatype, residue_index, ec_vector=None, motif_mask=None):
        """
        x_t:           [B, N, 14, 3]    噪声坐标
        t:             [B]              时间步
        atom_mask:     [B, N, 14]
        aatype:        [B, N]
        residue_index: [B, N]
        ec_vector:     [B, d_qwen]      Qwen3 向量 (某个层级截断的)
                                         level 0 时为零向量
        motif_mask:    [B, N]           (可选, scaffolding时用)
        
        返回: v_theta [B, N, 14, 3]
        """
        B, N = x_t.shape[:2]
        
        # 条件合并: time + EC
        t_embed = self.time_proj(self.time_embed(t))         # [B, d_cond]
        ec_embed = self.ec_proj(ec_vector)                    # [B, d_cond]
        # 注意: level 0 (无条件) 时 ec_vector 为零向量
        # 过瓶颈 MLP 后 ec_embed 也接近零, 相当于无 EC 条件
        cond = t_embed + ec_embed
        
        # motif_mask 作为额外特征 (给到 atom encoder 的输入中)
        # 具体做法: 在 atom_coords 上拼接 motif_mask
        # 让模型知道哪些残基是锚定的
        
        # 1. Atom Encoder
        token_repr, atom_repr = self.atom_encoder(x_t, atom_mask, aatype)
        
        # 2. Pair Init
        ca_coords = x_t[:, :, 1, :]  # Cα
        pair_repr = self.pair_init(token_repr, residue_index, ca_coords)
        
        # 3. Trunk
        for block in self.trunk:
            token_repr, pair_repr = block(token_repr, pair_repr, cond)
        
        # 4. Atom Decoder
        v_theta = self.atom_decoder(token_repr, atom_repr, atom_mask)
        
        return v_theta
```

## 五、Flow Matching 训练逻辑

### `flow/rectified_flow.py`

```python
class RectifiedFlowTrainer:
    """
    Rectified Flow Matching 的训练和采样逻辑。
    
    训练:
        1. 从数据集取真实结构 x_1 [B, N, 14, 3]
        2. 采样噪声 x_0 ~ N(0, I) [B, N, 14, 3]
        3. 采样时间 t ~ U(0, 1)  [B]
        4. 线性插值: x_t = (1-t)*x_0 + t*x_1
        5. 目标速度: v_target = x_1 - x_0
        6. 模型预测: v_pred = model(x_t, t, ...)
        7. Loss = MSE(v_pred, v_target), 仅在真实原子上计算
    
    采样:
        1. x_0 ~ N(0, I)
        2. for i in range(n_steps):
               t = i / n_steps
               v = model(x_t, t, ...)
               x_t = x_t + (1/n_steps) * v
        3. 返回 x_1_pred
    """
    
    def compute_loss(self, model, batch):
        x_1 = batch['coords']           # [B, N, 14, 3] 真实坐标
        atom_mask = batch['atom_mask']   # [B, N, 14]
        
        B = x_1.shape[0]
        
        # 噪声
        x_0 = torch.randn_like(x_1)
        
        # 时间
        t = torch.rand(B, device=x_1.device)
        
        # 插值
        t_expand = t[:, None, None, None]  # [B, 1, 1, 1]
        x_t = (1 - t_expand) * x_0 + t_expand * x_1
        
        # 目标
        v_target = x_1 - x_0
        
        # EC 条件: 层级 CFG dropout 已在 Dataset 中完成
        # Dataset.__getitem__ 随机选层级 0-5, 返回对应的 Qwen3 向量
        # level 0 时 ec_vector 为零向量, 自然实现无条件 dropout
        ec_vector = batch.get('ec_vector', None)
        # ec_level = batch.get('ec_level', 0)  # 可用于 logging/分析
        
        # 前向
        v_pred = model(x_t, t, atom_mask, batch['aatype'],
                       batch['residue_index'], ec_vector)
        
        # Loss: 仅在真实原子上计算
        mask = atom_mask.unsqueeze(-1)  # [B, N, 14, 1]
        loss = F.mse_loss(v_pred * mask, v_target * mask, reduction='sum')
        loss = loss / mask.sum()
        
        return loss
```

## 六、辅助 Loss

### `losses/aux_losses.py`

**仅靠 MSE 速度损失可能生成不满足化学约束的结构。加辅助 loss 保证物理合理性。**

```python
def bond_length_loss(pred_coords, atom_mask, aatype):
    """
    惩罚骨架键长偏离理想值。
    
    检查的键:
        N-Cα:  理想 1.458 Å
        Cα-C:  理想 1.523 Å
        C-O:   理想 1.231 Å
        C-N+1: 理想 1.329 Å (肽键, 连接相邻残基)
    
    Loss = mean(|d_pred - d_ideal|^2)
    """

def clash_loss(pred_coords, atom_mask, min_dist=1.5):
    """
    惩罚原子间距离过近 (空间位阻)。
    
    对所有非键合原子对:
        如果 d < min_dist: loss += (min_dist - d)^2
    
    注意: 同一残基内的原子不算 clash (它们是键合的)
    """

def total_aux_loss(pred_coords, atom_mask, aatype, weights):
    """
    加权求和:
        loss = w_bond * bond_length_loss + w_clash * clash_loss
    
    建议权重:
        w_bond = 0.5
        w_clash = 1.0
    """
```

## 七、Motif-Scaffolding 采样

### `tasks/motif_scaffolding.py`

```python
def sample_motif_scaffolding(
    model,
    motif_coords,       # [N_res, 14, 3]  motif区域的真实全原子坐标
    motif_mask,         # [N_res]  1=motif残基, 0=scaffold(待生成)
    atom_mask,          # [N_res, 14]
    ec_vectors_dict=None,  # {level: tensor[d_qwen]} 预计算的各层级向量
    n_steps=20,
    w_rxn=2.0,             # 化学反应引导强度
    w_full=3.0,            # 完整EC引导强度
):
    """
    通过 inpainting + 层级 CFG 实现 motif-scaffolding。
    
    算法:
        1. scaffold 区域从噪声开始, motif 区域从对应 t 的插值位置开始
        2. 每一步:
           a. 对 motif 区域计算当前时间 t 对应的插值位置
           b. 拼接 motif(插值) + scaffold(当前去噪结果)
           c. 模型预测速度场 (层级 CFG: 无条件 / rxn-only / full)
           d. 仅更新 scaffold 区域
        3. 最终: motif 用真实坐标, scaffold 用生成结果
    """
    device = motif_coords.device
    N_res = motif_coords.shape[0]
    
    motif_mask_expand = motif_mask[:, None, None]      # [N, 1, 1]
    scaffold_mask = 1.0 - motif_mask_expand
    
    # 固定的噪声 (对应 motif 在 t=0 时的状态)
    noise_motif = torch.randn_like(motif_coords)
    
    # scaffold 初始化为纯噪声
    x_scaffold = torch.randn_like(motif_coords)
    
    dt = 1.0 / n_steps
    
    for step in range(n_steps):
        t = step * dt
        t_tensor = torch.tensor([t], device=device)
        
        # motif 在时间 t 的位置 (线性插值)
        motif_t = (1 - t) * noise_motif + t * motif_coords
        
        # 拼接
        x_t = motif_mask_expand * motif_t + scaffold_mask * x_scaffold
        
        # 层级 CFG (3次前向)
        if ec_vectors_dict is not None and (w_rxn > 0 or w_full > 0):
            x_t_batch = x_t.unsqueeze(0)
            mask_batch = atom_mask.unsqueeze(0)
            
            # 无条件
            v_uncond = model(x_t_batch, t_tensor, mask_batch,
                           ..., ec_vector=ec_vectors_dict[0].unsqueeze(0)).squeeze(0)
            # rxn-only (level 1)
            v_rxn = model(x_t_batch, t_tensor, mask_batch,
                         ..., ec_vector=ec_vectors_dict[1].unsqueeze(0)).squeeze(0)
            # 完整 (level 5)
            v_full = model(x_t_batch, t_tensor, mask_batch,
                          ..., ec_vector=ec_vectors_dict[5].unsqueeze(0)).squeeze(0)
            
            # 层级引导叠加
            v = v_uncond + w_rxn * (v_rxn - v_uncond) + w_full * (v_full - v_rxn)
        else:
            v = model(x_t.unsqueeze(0), t_tensor, ...).squeeze(0)
        
        # 仅更新 scaffold
        x_scaffold = x_scaffold + dt * v * scaffold_mask.squeeze(-1).unsqueeze(-1)
    
    # 最终结果
    x_final = motif_mask_expand * motif_coords + scaffold_mask * x_scaffold
    
    return x_final
```

## 七-B、层级 Classifier-Free Guidance

### `flow/hierarchical_cfg.py`

**层级 CFG 是本方案的核心创新点之一。** 不同于标准 CFG 的"有条件 vs 无条件"二元选择，这里将 EC 条件分解为多个粒度层级，推理时可以分别控制每个层级的引导强度。

```python
"""
EC 条件层级设计:

Level 0 (无条件):     ""
Level 1 (rxn-only):   "Reactions: primary alcohol + NAD(+) = aldehyde + NADH + H(+)"
Level 2 (lv1+rxn):    "Oxidoreductases | Reactions: ..."
Level 3 (lv1+2+rxn):  "Oxidoreductases | Acting on CH-OH | Reactions: ..."
Level 4 (lv1+2+3+rxn):"Oxidoreductases | Acting on CH-OH | With NAD(+) | Reactions: ..."
Level 5 (full):        "Oxidoreductases | Acting on CH-OH | With NAD(+) | alcohol dehydrogenase | Reactions: ..."

核心设计:
  - rxn 始终保留: 反应方程式是最硬的化学约束
  - EC 层级逐步叠加: 提供从粗到细的折叠先验
  - 两个旋钮: w_rxn (化学反应引导) 和 w_full (EC层级精细引导)
"""

import torch


class HierarchicalCFGSampler:
    """
    层级 CFG 采样器。
    
    支持三种模式:
    1. 简单模式 (3次前向):  无条件 + rxn-only + full
    2. 完整模式 (6次前向):  每个层级都计算
    3. 自定义模式:          任意选择 2-3 个层级组合
    """
    
    def __init__(self, model, ec_vectors_dict):
        """
        model:           训练好的 AllAtomFlowModel
        ec_vectors_dict: {level: tensor[d_qwen]}  预计算的各层级 Qwen3 向量
                         keys: 0, 1, 2, 3, 4, 5
        """
        self.model = model
        self.ec_vecs = ec_vectors_dict
    
    def predict_velocity(self, x_t, t, atom_mask, aatype, residue_index,
                         w_rxn=2.0, w_full=3.0, motif_mask=None):
        """
        简单模式: 3 次前向，2 级引导。
        
        v = v_uncond + w_rxn * (v_rxn - v_uncond) + w_full * (v_full - v_rxn)
        
        w_rxn: 控制 "生成的蛋白质能催化这个反应" 的强度
        w_full: 控制 "生成的蛋白质属于这个具体EC分类" 的强度
        
        典型用法:
          w_rxn=2.0, w_full=3.0  → 标准引导
          w_rxn=3.0, w_full=0.0  → 只用反应约束, 不限制折叠类型
          w_rxn=0.0, w_full=0.0  → 无条件生成
          w_rxn=1.0, w_full=5.0  → 弱反应约束, 强EC类别约束
        """
        args = (x_t, t, atom_mask, aatype, residue_index)
        kwargs = {'motif_mask': motif_mask}
        
        # 三次前向
        v_uncond = self.model(*args, ec_vector=self.ec_vecs[0].expand(x_t.shape[0], -1), **kwargs)
        v_rxn    = self.model(*args, ec_vector=self.ec_vecs[1].expand(x_t.shape[0], -1), **kwargs)
        v_full   = self.model(*args, ec_vector=self.ec_vecs[5].expand(x_t.shape[0], -1), **kwargs)
        
        # 层级叠加
        v = v_uncond + w_rxn * (v_rxn - v_uncond) + w_full * (v_full - v_rxn)
        
        return v
    
    def predict_velocity_full(self, x_t, t, atom_mask, aatype, residue_index,
                              weights=(1.0, 1.0, 1.5, 2.0, 2.5)):
        """
        完整模式: 6 次前向，5 级引导。
        
        v = v_0 + w1*(v1-v0) + w2*(v2-v1) + w3*(v3-v2) + w4*(v4-v3) + w5*(v5-v4)
        
        weights: (w1, w2, w3, w4, w5) 对应 level 1-5 的增量引导强度
        
        一般不需要用这个模式, 简单模式足够。
        仅在需要精细控制每个 EC 层级的引导强度时使用。
        """
        args = (x_t, t, atom_mask, aatype, residue_index)
        B = x_t.shape[0]
        
        vs = {}
        for level in range(6):
            ec_vec = self.ec_vecs[level].expand(B, -1)
            vs[level] = self.model(*args, ec_vector=ec_vec)
        
        v = vs[0]
        for level in range(1, 6):
            v = v + weights[level - 1] * (vs[level] - vs[level - 1])
        
        return v


def sample_unconditional(model, n_res, atom_mask, n_steps=20, device='cuda'):
    """无条件生成 (baseline, 不使用任何 EC 条件)"""
    x_t = torch.randn(1, n_res, 14, 3, device=device)
    dt = 1.0 / n_steps
    
    for step in range(n_steps):
        t = torch.tensor([step * dt], device=device)
        ec_zero = torch.zeros(1, 1024, device=device)  # d_qwen
        v = model(x_t, t, atom_mask.unsqueeze(0), ..., ec_vector=ec_zero)
        x_t = x_t + dt * v.clamp(-10, 10)
    
    return x_t.squeeze(0)


def sample_with_hierarchical_cfg(
    model, ec_vectors_dict, n_res, atom_mask,
    aatype, residue_index,
    n_steps=20, w_rxn=2.0, w_full=3.0,
    device='cuda'
):
    """
    带层级 CFG 的完整采样流程。
    
    参数:
        ec_vectors_dict: {0: zero_vec, 1: rxn_vec, ..., 5: full_vec}
        w_rxn:  化学反应引导强度 (建议 1.0-3.0)
        w_full: EC 层级精细引导强度 (建议 1.0-4.0)
    """
    sampler = HierarchicalCFGSampler(model, ec_vectors_dict)
    
    x_t = torch.randn(1, n_res, 14, 3, device=device)
    dt = 1.0 / n_steps
    
    for step in range(n_steps):
        t = torch.tensor([step * dt], device=device)
        
        v = sampler.predict_velocity(
            x_t, t, atom_mask.unsqueeze(0),
            aatype.unsqueeze(0), residue_index.unsqueeze(0),
            w_rxn=w_rxn, w_full=w_full
        )
        
        x_t = x_t + dt * v.clamp(-10, 10)
    
    return x_t.squeeze(0)
```

### 训练时 Loss 计算

```python
def compute_loss(self, model, batch):
    x_1 = batch['coords']           # [B, N, 14, 3]
    atom_mask = batch['atom_mask']   # [B, N, 14]
    motif_mask = batch['motif_mask'] # [B, N]
    
    # 噪声 + 时间 + 插值 (标准 rectified flow)
    x_0 = torch.randn_like(x_1)
    t = torch.rand(B, device=x_1.device)
    t_expand = t[:, None, None, None]
    x_t = (1 - t_expand) * x_0 + t_expand * x_1
    v_target = x_1 - x_0
    
    # motif inpainting: motif 区域用真实插值位置替换
    # (训练时也做 replacement, 让模型学会 "motif 区域是锚定的")
    motif_expand = motif_mask[:, :, None, None]  # [B, N, 1, 1]
    motif_t = (1 - t_expand) * torch.randn_like(x_1) + t_expand * x_1
    x_t = motif_expand * motif_t + (1 - motif_expand) * x_t
    
    # 模型前向 (EC 条件已由 Dataset 处理好层级 dropout)
    v_pred = model(x_t, t, atom_mask, batch['aatype'],
                   batch['residue_index'], batch['ec_vector'],
                   motif_mask=motif_mask)
    
    # Loss: 仅在真实原子上计算, motif 和 scaffold 都参与
    mask = atom_mask.unsqueeze(-1)  # [B, N, 14, 1]
    flow_loss = (v_pred * mask - v_target * mask).pow(2).sum() / mask.sum()
    
    # 辅助 loss (在预测的 x_1 = x_t + (1-t)*v_pred 上计算)
    x_1_pred = x_t + (1 - t_expand) * v_pred
    aux_loss = bond_length_loss(x_1_pred, atom_mask) * 0.5 \
             + clash_loss(x_1_pred, atom_mask) * 1.0
    
    return flow_loss + aux_loss
```

### Motif 选取策略

```
随机 motif (基础):
  - 均匀随机选 2-8 个残基, 不关心它们是什么
  - 优点: 简单, 让模型学会泛化到任意 motif 位置
  - 这是训练时的默认做法

活性位点 motif (可选, 增强版):

推理时:
  - 你指定具体的 motif 残基及其全原子坐标
  - 模型为这些 motif 生成包裹它们的 scaffold
```

## 十一、关键实现注意事项

### 必须做对的事

```
1. SE(3) 数据增强:
   - 每个样本、每个 epoch 都要随机旋转平移
   - 先做质心归零再旋转 (不归零旋转后坐标值会很大)
   - 旋转矩阵必须是均匀采样 SO(3) (不是随机 Euler 角)

2. 虚拟原子处理:
   - 缺失原子位置填 Cα 坐标
   - Loss 计算时 mask 掉虚拟原子 (不参与梯度)
   - Decoder 输出后对虚拟原子的速度置零

3. Pair representation 中的距离:
   - 必须用 x_t (噪声坐标) 计算距离, 不是用 x_1 (真实坐标)
   - 否则模型会在 pair 中偷看答案

4. 时间步编码:
   - t ∈ [0, 1], 用正弦位置编码
   - 不要离散化为整数步, 保持连续

5. 梯度裁剪:
   - grad_clip = 1.0, 蛋白质模型训练容易梯度爆炸

6. EMA:
   - 训练时维护 EMA 权重 (decay=0.999)
   - 推理和评估全部用 EMA 权重

7. 混合精度:
   - 用 BF16 而不是 FP16 (FP16 在蛋白质坐标范围容易溢出)
```

### 常见坑

```
1. Padding 导致 loss 计算错误:
   - batch 内不同长度的蛋白质需要 padding
   - Loss 的 reduction 必须手动处理: 先 sum(loss * mask), 再除以 mask.sum()
   - 不能用 F.mse_loss(reduction='mean'), 因为 mean 会包含 padding 位置

2. 相对位置编码的链间处理:
   - 如果训练数据有多链蛋白, 不同链之间的 rel_pos 应该设为一个大值 (如 999)
   - 或者只用单链蛋白训练, 避免链间处理的复杂度

3. 推理时的数值稳定性:
   - ODE Euler 步进时, 如果 v_pred 太大, 坐标可能飞走
   - 加 clamp: x_t = x_t + dt * v.clamp(-10, 10)

4. CFG 权重过大导致崩溃:
   - w_rxn, w_full 从 1.0 开始试, 不要上来就 4.0
   - v_cond 和 v_uncond 差太大时 cfg 会放大噪声

5. 数据泄漏:
   - aatype 在无条件生成时不应该作为输入
   - 仅在 motif-scaffolding 时, motif 区域的 aatype 才输入
   - scaffold 区域的 aatype 用 mask token (索引 20)

6. 层级 CFG 相关:
   - 预计算 EC 向量时注意文本去重, 同一个 lv1 文本不要重复编码
   - level 0 (无条件) 的向量必须是全零, 不是空字符串的 Qwen3 编码
     (空字符串编码出来不是零向量, 会引入不需要的信号)
   - 推理时 w_rxn 和 w_full 从小到大调: 先 w_rxn=1.0, w_full=1.0
     逐步增加, 观察结构质量和多样性的变化
   - 如果某个 EC 的反应方程式为空 (数据缺失), 该 EC 的 level 1-4
     退化为仅有 EC 层级信息, rxn 引导无效 → 需要检查数据完整性
```
